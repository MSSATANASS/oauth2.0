# Crypto Exchange Manager (OAuth 2.0)

Securely manage your Gemini, Binance, Coinbase, Kraken, and Bitget accounts in one place.

## Features
- **OAuth 2.0 Integration**: Connect securely with major exchanges.
- **Supabase Auth**: Secure user management and sessions.
- **Encrypted Storage**: API keys are encrypted before storage.
- **Real-time Notifications**: Telegram integration for connection alerts.
